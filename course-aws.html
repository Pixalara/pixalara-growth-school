AWS Cloud Architect

Become a Certified AWS Solutions Architect (Associate & Professional).

Master the AWS Cloud

This course covers the full spectrum of AWS services. You will learn to design scalable, reliable, and secure cloud applications directly from industry experts.

Certification Ready

Prepares you for the AWS Solutions Architect Associate exam (SAA-C03).

Real Infrastructure

Build Multi-Tier Architectures, VPCs, and Serverless apps.

Module 1: Identity & Access Management (IAM)
  • IAM Users, Groups, and Roles fundamentals
  • Creating and managing IAM Policies (JSON)
  • Enforcing Multi-Factor Authentication (MFA)
  • Designing secure access for applications and teams
  • Permission Boundaries and Best Practices
Module 2: Compute (EC2 & Auto Scaling)
  • EC2 Instance Types & Pricing Models
  • Creating custom Amazon Machine Images (AMIs)
  • Configuring Security Groups and Elastic IPs
  • Setting up Auto Scaling Groups (ASG)
  • Load Balancing (ALB, NLB, GWLB)
Module 3: Storage (S3, EBS, EFS)
  • S3 Object Storage: Versioning, Encryption, Security
  • S3 Lifecycle Rules & Glacier Archival
  • EBS Volume types (GP3, IO2) & Snapshots
  • EFS: Shared File Storage for Linux
  • Storage Gateway & Snow Family overview
Module 4: Networking (VPC & Route 53)
  • Designing custom VPCs & Subnets (Public/Private)
  • NAT Gateways & Internet Gateways
  • Network ACLs (NACL) vs Security Groups
  • VPC Peering & Transit Gateway
  • DNS Management with Route 53
Module 5: Serverless & Databases
  • Event-driven architecture with AWS Lambda
  • API Gateway & Step Functions
  • NoSQL design with DynamoDB
  • Relational DBs: RDS, Aurora & Multi-AZ
  • ElastiCache (Redis/Memcached)
Module 6: Containers (ECS & EKS)
  • Amazon ECS (Elastic Container Service)
  • AWS Fargate (Serverless Containers)
  • Amazon EKS (Managed Kubernetes)
  • Container Registry (ECR)
Module 7: Monitoring & Logging
  • CloudWatch: Metrics, Alarms, and Dashboards
  • CloudWatch Logs & Events
  • Auditing API activity with CloudTrail
  • Tracing distributed apps with AWS X-Ray
Module 8: Security & Encryption
  • KMS: Key Management & Encryption
  • Secrets Manager & Parameter Store
  • WAF (Web Application Firewall) & Shield
  • GuardDuty & Inspector
Module 9: Architecture Design
  • Designing Highly Available & Fault-Tolerant systems
  • Disaster Recovery (Backup, Pilot Light, Multi-Region)
  • Microservices & Event-Driven Patterns
  • Cost-Optimized Architecture
Module 10: Deployment & Automation
  • Infrastructure as Code (IaC) with CloudFormation
  • Introduction to AWS CDK
  • CI/CD with CodePipeline & CodeBuild
  • GitHub Actions Integration
Module 11: Cost Optimization
  • AWS Cost Explorer & Budgets
  • Savings Plans vs Reserved Instances
  • Cost Allocation Tags
  • Trusted Advisor recommendations
Module 12: Exam Prep & Practice
  • SAA-C03 Exam Strategy & Tips
  • Solving Scenario-based Questions
  • Architecture Case Studies review
  • 4 Full-Length Practice Mock Tests
